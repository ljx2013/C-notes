
# C++ 进阶：输入与输出（scanf/printf）

## 一、概述

在 C++ 中，除了使用 C++ 风格的流操作（`cin`/`cout`）外，还可以使用 C 语言风格的 `scanf` 和 `printf` 函数。这两种函数在某些场景下更加高效和灵活，特别是在需要精确控制输入输出格式时。

### 1.1 头文件

使用 `scanf` 和 `printf` 需要包含 C 标准库头文件：

```cpp
#include <cstdio>  // C++ 风格包含
// 或
#include <stdio.h> // C 风格包含
```

### 1.2 性能对比

| 特性 | cin/cout | scanf/printf |
|------|----------|--------------|
| 速度 | 较慢（默认同步） | 较快 |
| 格式控制 | 较复杂 | 简洁灵活 |
| 类型安全 | 是 | 否 |
| 易用性 | 高 | 中等 |

## 二、printf 函数详解

### 2.1 基本语法

```cpp
printf("格式字符串", 参数1, 参数2, ...);
```

### 2.2 格式说明符

格式说明符以 `%` 开头，用于指定输出的格式：

| 格式符 | 含义 | 示例 |
|--------|------|------|
| `%d` | 十进制整数 | 42 |
| `%i` | 十进制整数（同 %d） | 42 |
| `%u` | 无符号十进制整数 | 42 |
| `%o` | 八进制整数 | 52 |
| `%x` | 十六进制整数（小写） | 2a |
| `%X` | 十六进制整数（大写） | 2A |
| `%f` | 浮点数（默认6位小数） | 3.141593 |
| `%e` | 科学计数法（小写） | 3.141593e+00 |
| `%E` | 科学计数法（大写） | 3.141593E+00 |
| `%g` | 自动选择 %f 或 %e | 3.14159 |
| `%c` | 单个字符 | A |
| `%s` | 字符串 | Hello |
| `%p` | 指针地址 | 0x7fff5fbff8ac |
| `%%` | 输出百分号 | % |

### 2.3 基本用法示例

```cpp
#include <cstdio>

int main() {
    int num = 42;
    double pi = 3.1415926;
    char ch = 'A';
    const char* str = "Hello";
    
    printf("Integer: %d\n", num);
    printf("Double: %f\n", pi);
    printf("Character: %c\n", ch);
    printf("String: %s\n", str);
    
    return 0;
}
```

### 2.4 宽度和精度控制

#### 设置宽度

```cpp
printf("%5d\n", 42);      // 输出: "   42"（总宽度5，右对齐）
printf("%-5d\n", 42);     // 输出: "42   "（总宽度5，左对齐）
printf("%10s\n", "Hi");   // 输出: "        Hi"
```

#### 设置精度

```cpp
printf("%.2f\n", 3.14159);  // 输出: 3.14（保留2位小数）
printf("%.5s\n", "Hello");  // 输出: Hello（最多5个字符）
printf("%10.2f\n", 3.14159); // 输出: "      3.14"（宽度10，精度2）
```

### 2.5 综合示例

```cpp
#include <cstdio>

int main() {
    printf("|%5d|\n", 123);     // |  123|
    printf("|%-5d|\n", 123);    // |123  |
    printf("|%05d|\n", 123);    // |00123|（前导零填充）
    printf("|%+d|\n", 123);     // |+123|（显示符号）
    printf("|% d|\n", 123);     // | 123|（空格代替正号）
    
    double num = 12345.6789;
    printf("|%10.2f|\n", num);  // |  12345.68|
    printf("|%e|\n", num);      // |1.234568e+04|
    printf("|%g|\n", num);      // |12345.7|
    
    return 0;
}
```

## 三、scanf 函数详解

### 3.1 基本语法

```cpp
scanf("格式字符串", &变量1, &变量2, ...);
```

**注意**：变量前必须加 `&` 取地址符（数组和指针除外）。

### 3.2 格式说明符

| 格式符 | 含义 | 对应类型 |
|--------|------|----------|
| `%d` | 十进制整数 | int* |
| `%i` | 整数（自动识别进制） | int* |
| `%u` | 无符号整数 | unsigned int* |
| `%o` | 八进制整数 | int* |
| `%x` | 十六进制整数 | int* |
| `%f` | 浮点数 | float* |
| `%lf` | 双精度浮点数 | double* |
| `%c` | 单个字符 | char* |
| `%s` | 字符串 | char* |
| `%p` | 指针地址 | void** |

### 3.3 基本用法示例

```cpp
#include <cstdio>

int main() {
    int num;
    double pi;
    char ch;
    char str[100];
    
    printf("Enter an integer: ");
    scanf("%d", &num);
    
    printf("Enter a double: ");
    scanf("%lf", &pi);
    
    printf("Enter a character: ");
    scanf(" %c", &ch);  // 注意空格，跳过空白字符
    
    printf("Enter a string: ");
    scanf("%s", str);   // 数组名本身就是地址，无需 &
    
    printf("num=%d, pi=%.2f, ch=%c, str=%s\n", num, pi, ch, str);
    
    return 0;
}
```

### 3.4 输入宽度限制

```cpp
#include <cstdio>

int main() {
    char str[10];
    scanf("%9s", str);  // 最多读取9个字符，留一个位置给 '\0'
    printf("You entered: %s\n", str);
    return 0;
}
```

### 3.5 混合输入

```cpp
#include <cstdio>

int main() {
    int day, month, year;
    printf("Enter date (dd/mm/yyyy): ");
    scanf("%d/%d/%d", &day, &month, &year);
    printf("Date: %02d-%02d-%04d\n", day, month, year);
    return 0;
}
```

### 3.6 返回值

`scanf` 返回成功读取的变量个数：

```cpp
#include <cstdio>

int main() {
    int a, b;
    printf("Enter two numbers: ");
    int result = scanf("%d %d", &a, &b);
    
    if (result == 2) {
        printf("Both numbers read successfully\n");
    } else if (result == 1) {
        printf("Only one number read\n");
    } else {
        printf("No numbers read\n");
    }
    
    return 0;
}
```

## 四、高级用法

### 4.1 printf 的标志位

| 标志 | 作用 | 示例 |
|------|------|------|
| `-` | 左对齐 | `%-5d` |
| `+` | 显示正负号 | `%+d` |
| ` ` | 正数前加空格 | `% d` |
| `0` | 前导零填充 | `%05d` |
| `#` | 显示进制前缀 | `%#x` → 0x |

### 4.2 可变参数示例

```cpp
#include <cstdio>

void printNumbers(int count, ...) {
    va_list args;
    va_start(args, count);
    
    for (int i = 0; i < count; i++) {
        int num = va_arg(args, int);
        printf("%d ", num);
    }
    
    va_end(args);
    printf("\n");
}

int main() {
    printNumbers(3, 10, 20, 30);  // 输出: 10 20 30
    printNumbers(5, 1, 2, 3, 4, 5); // 输出: 1 2 3 4 5
    return 0;
}
```

### 4.3 格式化到字符串（sprintf）

```cpp
#include <cstdio>

int main() {
    char buffer[100];
    int num = 42;
    double pi = 3.14159;
    
    sprintf(buffer, "Number: %d, Pi: %.2f", num, pi);
    printf("%s\n", buffer);  // 输出: Number: 42, Pi: 3.14
    
    return 0;
}
```

### 4.4 从字符串读取（sscanf）

```cpp
#include <cstdio>

int main() {
    const char* str = "John 25 175.5";
    char name[20];
    int age;
    double height;
    
    sscanf(str, "%s %d %lf", name, &age, &height);
    printf("Name: %s, Age: %d, Height: %.2f\n", name, age, height);
    
    return 0;
}
```

## 五、常见问题

### 5.1 scanf 跳过空白字符

`scanf` 默认会跳过空白字符（空格、换行、制表符）：

```cpp
// 输入: "  42  "
int num;
scanf("%d", &num);  // 正确读取 42
```

### 5.2 读取字符时的问题

```cpp
int num;
char ch;

scanf("%d", &num);
scanf("%c", &ch);  // 读取到的是换行符 '\n'

// 解决方案：
scanf("%d", &num);
scanf(" %c", &ch); // 在 %c 前加空格
```

### 5.3 缓冲区溢出风险

```cpp
char str[10];
scanf("%s", str);  // 危险！如果输入超过9个字符会溢出

// 安全做法：
scanf("%9s", str);  // 限制最多读取9个字符
```

## 六、cin/cout 与 scanf/printf 对比

### 6.1 性能测试

```cpp
#include <cstdio>
#include <iostream>
#include <time.h>

const int N = 1000000;

int main() {
    // printf 测试
    clock_t start = clock();
    for (int i = 0; i < N; i++) {
        printf("%d\n", i);
    }
    printf("printf time: %f\n", (double)(clock() - start) / CLOCKS_PER_SEC);
    
    // cout 测试（关闭同步）
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);
    
    start = clock();
    for (int i = 0; i < N; i++) {
        std::cout << i << '\n';
    }
    printf("cout time: %f\n", (double)(clock() - start) / CLOCKS_PER_SEC);
    
    return 0;
}
```

### 6.2 选择建议

| 场景 | 推荐方案 |
|------|----------|
| 大量数据输出 | printf（更快） |
| 需要精确格式控制 | printf |
| 类型安全重要 | cin/cout |
| 简单的输入输出 | 两者均可 |
| 混合使用 | 注意同步问题 |

## 七、实战示例

### 示例1：格式化表格输出

```cpp
#include <cstdio>

int main() {
    printf("+--------+--------+--------+\n");
    printf("|  Name  |  Age   |  Score |\n");
    printf("+--------+--------+--------+\n");
    printf("| %-6s | %6d | %6.1f |\n", "Alice", 20, 95.5);
    printf("| %-6s | %6d | %6.1f |\n", "Bob", 22, 88.0);
    printf("| %-6s | %6d | %6.1f |\n", "Charlie", 19, 92.3);
    printf("+--------+--------+--------+\n");
    return 0;
}
```

### 示例2：解析命令行参数

```cpp
#include <cstdio>

int main(int argc, char* argv[]) {
    if (argc != 3) {
        printf("Usage: %s <width> <height>\n", argv[0]);
        return 1;
    }
    
    int width, height;
    if (sscanf(argv[1], "%d", &width) != 1 || 
        sscanf(argv[2], "%d", &height) != 1) {
        printf("Invalid arguments\n");
        return 1;
    }
    
    printf("Area: %d\n", width * height);
    return 0;
}
```

## 八、总结

| 函数 | 功能 | 特点 |
|------|------|------|
| `printf` | 格式化输出 | 快速、灵活、格式控制强 |
| `scanf` | 格式化输入 | 快速、但需要注意地址符和缓冲区 |
| `sprintf` | 格式化到字符串 | 生成格式化字符串 |
| `sscanf` | 从字符串读取 | 解析字符串中的数据 |

掌握 `scanf` 和 `printf` 是 C++ 程序员的重要技能，在性能敏感和需要精确格式控制的场景中非常有用。
