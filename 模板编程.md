
# C++ 模板编程

## 一、模板概述

模板是 C++ 的泛型编程工具，允许编写与类型无关的代码，提高代码复用性。

### 1.1 模板分类

- **函数模板**：创建通用函数
- **类模板**：创建通用类

## 二、函数模板

### 2.1 基本语法

```cpp
template <typename T>
T function_name(T parameter) {
    // 函数体
}
```

### 2.2 示例：通用交换函数

```cpp
template <typename T>
void swap(T& a, T& b) {
    T temp = a;
    a = b;
    b = temp;
}

// 使用
int x = 5, y = 10;
swap(x, y);

double a = 3.14, b = 2.71;
swap(a, b);
```

### 2.3 多参数模板

```cpp
template <typename T, typename U>
auto add(T a, U b) {
    return a + b;
}

// 使用
auto result = add(5, 3.14);  // 返回 double
```

### 2.4 模板特化

```cpp
// 通用模板
template <typename T>
T max(T a, T b) {
    return a > b ? a : b;
}

// 针对 const char* 的特化
template <>
const char* max<const char*>(const char* a, const char* b) {
    return strcmp(a, b) > 0 ? a : b;
}
```

## 三、类模板

### 3.1 基本语法

```cpp
template <typename T>
class ClassName {
public:
    ClassName(T value);
    T getValue() const;
private:
    T data;
};
```

### 3.2 示例：通用容器类

```cpp
template <typename T>
class Box {
private:
    T item;
public:
    Box(T i) : item(i) {}
    T getItem() const { return item; }
    void setItem(T i) { item = i; }
};

// 使用
Box<int> intBox(42);
Box<std::string> strBox("Hello");
```

### 3.3 类模板成员函数定义

```cpp
template <typename T>
class MyClass {
public:
    void display(T value);
};

// 成员函数在类外定义
template <typename T>
void MyClass<T>::display(T value) {
    std::cout << value << std::endl;
}
```

## 四、模板参数

### 4.1 类型参数

```cpp
template <typename T>  // T 是类型参数
T process(T value) { ... }
```

### 4.2 非类型参数

```cpp
template <typename T, int size>
class Array {
private:
    T arr[size];
public:
    T& operator[](int index) { return arr[index]; }
};

// 使用
Array<int, 5> arr;  // size = 5
```

### 4.3 默认模板参数

```cpp
template <typename T = int>
class Container {
private:
    T data;
public:
    Container(T d = T()) : data(d) {}
};

// 使用
Container<> c1;      // T = int
Container<double> c2; // T = double
```

## 五、模板进阶特性

### 5.1 模板别名

```cpp
template <typename T>
using Vec = std::vector<T>;

// 使用
Vec<int> numbers;  // 等价于 std::vector<int>
```

### 5.2 可变参数模板

```cpp
// 递归终止条件
void print() {}

// 可变参数模板
template <typename T, typename... Args>
void print(T first, Args... args) {
    std::cout << first << " ";
    print(args...);  // 递归调用
}

// 使用
print(1, "hello", 3.14, 'A');
```

### 5.3 折叠表达式（C++17）

```cpp
template <typename... Args>
auto sum(Args... args) {
    return (... + args);  // 折叠求和
}

// 使用
sum(1, 2, 3, 4);  // 返回 10
```

## 六、模板特化

### 6.1 类模板特化

```cpp
template <typename T>
class MyClass {
public:
    void display() { std::cout << "Generic\n"; }
};

// 特化
template <>
class MyClass<int> {
public:
    void display() { std::cout << "Specialized for int\n"; }
};
```

### 6.2 部分特化

```cpp
template <typename T, typename U>
class Pair {
public:
    void display() { std::cout << "Generic Pair\n"; }
};

// 部分特化
template <typename T>
class Pair<T, T> {
public:
    void display() { std::cout << "Pair with same types\n"; }
};
```

## 七、SFINAE

### 7.1 概念

SFINAE（Substitution Failure Is Not An Error）：替换失败不是错误。

```cpp
#include <type_traits>

template <typename T>
typename std::enable_if<std::is_integral<T>::value, T>::type
add(T a, T b) {
    return a + b;
}

template <typename T>
typename std::enable_if<std::is_floating_point<T>::value, T>::type
add(T a, T b) {
    return a + b;
}
```

### 7.2 constexpr if（C++17）

```cpp
template <typename T>
void process(T value) {
    if constexpr (std::is_integral_v<T>) {
        std::cout << "Integer: " << value << std::endl;
    } else if constexpr (std::is_floating_point_v<T>) {
        std::cout << "Float: " << value << std::endl;
    }
}
```

## 八、实战示例

### 示例1：通用栈模板

```cpp
#include <iostream>
#include <vector>
using namespace std;

template <typename T>
class Stack {
private:
    vector<T> elements;
public:
    void push(T element) {
        elements.push_back(element);
    }
    
    T pop() {
        T top = elements.back();
        elements.pop_back();
        return top;
    }
    
    bool empty() const {
        return elements.empty();
    }
};

int main() {
    Stack<int> intStack;
    intStack.push(1);
    intStack.push(2);
    cout << intStack.pop() << endl;  // 2
    
    Stack<string> strStack;
    strStack.push("Hello");
    strStack.push("World");
    cout << strStack.pop() << endl;  // World
    
    return 0;
}
```

### 示例2：类型萃取

```cpp
#include <iostream>
#include <type_traits>
using namespace std;

template <typename T>
void printTypeInfo() {
    cout << "Type: " << typeid(T).name() << endl;
    cout << "Is integral: " << is_integral<T>::value << endl;
    cout << "Is pointer: " << is_pointer<T>::value << endl;
    cout << "Is reference: " << is_reference<T>::value << endl;
}

int main() {
    printTypeInfo<int>();
    printTypeInfo<double>();
    printTypeInfo<int*>();
    return 0;
}
```

## 九、总结

| 概念 | 说明 |
|------|------|
| 函数模板 | 创建类型无关的函数 |
| 类模板 | 创建类型无关的类 |
| 类型参数 | typename T |
| 非类型参数 | 常量表达式 |
| 默认参数 | 模板参数默认值 |
| 特化 | 针对特定类型的定制实现 |
| SFINAE | 替换失败不是错误 |
| 折叠表达式 | 可变参数包的展开 |

模板是 C++ 泛型编程的核心，掌握模板编程可以编写高度复用的代码。
