
# C++ 继承与多态

## 一、继承概述

继承是面向对象编程的核心特性之一，允许一个类（子类/派生类）获取另一个类（父类/基类）的属性和方法。

### 1.1 继承的优点

- 代码复用
- 层次化组织
- 易于扩展

## 二、继承的基本语法

### 2.1 定义派生类

```cpp
class BaseClass {
    // 基类成员
};

class DerivedClass : access_specifier BaseClass {
    // 派生类成员
};
```

### 2.2 访问说明符

| 继承方式 | 基类 public | 基类 protected | 基类 private |
|----------|-------------|----------------|--------------|
| public | public | protected | 不可访问 |
| protected | protected | protected | 不可访问 |
| private | private | private | 不可访问 |

### 2.3 示例：简单继承

```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
protected:
    string name;
    int age;

public:
    Person(string n, int a) : name(n), age(a) {}
    
    void introduce() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

class Student : public Person {
private:
    int studentId;

public:
    Student(string n, int a, int id) : Person(n, a), studentId(id) {}
    
    void study() {
        cout << name << " is studying" << endl;
    }
};
```

## 三、构造函数与析构函数的执行顺序

### 3.1 构造顺序

1. 基类构造函数
2. 成员对象构造函数
3. 派生类构造函数

### 3.2 析构顺序（与构造相反）

1. 派生类析构函数
2. 成员对象析构函数
3. 基类析构函数

```cpp
class Base {
public:
    Base() { cout << "Base constructor\n"; }
    ~Base() { cout << "Base destructor\n"; }
};

class Derived : public Base {
public:
    Derived() { cout << "Derived constructor\n"; }
    ~Derived() { cout << "Derived destructor\n"; }
};

// 输出：
// Base constructor
// Derived constructor
// Derived destructor
// Base destructor
```

## 四、多态

### 4.1 多态的概念

多态允许不同类的对象对同一消息做出不同的响应。

### 4.2 虚函数

虚函数是实现多态的关键。

```cpp
class Shape {
public:
    virtual double getArea() {
        return 0;
    }
};

class Circle : public Shape {
private:
    double radius;

public:
    Circle(double r) : radius(r) {}
    
    double getArea() override {
        return 3.14159 * radius * radius;
    }
};

class Rectangle : public Shape {
private:
    double width, height;

public:
    Rectangle(double w, double h) : width(w), height(h) {}
    
    double getArea() override {
        return width * height;
    }
};
```

### 4.3 纯虚函数与抽象类

```cpp
class Shape {
public:
    virtual double getArea() = 0;  // 纯虚函数
};
```

包含纯虚函数的类是抽象类，不能实例化。

## 五、虚析构函数

### 5.1 为什么需要虚析构函数

```cpp
class Base {
public:
    virtual ~Base() {  // 虚析构函数
        cout << "Base destructor\n";
    }
};

class Derived : public Base {
public:
    ~Derived() {
        cout << "Derived destructor\n";
    }
};

// 使用
Base* ptr = new Derived();
delete ptr;  // 正确调用两个析构函数
```

## 六、静态多态与动态多态

### 6.1 静态多态（编译时多态）

通过函数重载和模板实现。

```cpp
int add(int a, int b) { return a + b; }
double add(double a, double b) { return a + b; }
```

### 6.2 动态多态（运行时多态）

通过虚函数实现。

```cpp
Shape* shapes[] = {new Circle(5), new Rectangle(4, 6)};
for (int i = 0; i < 2; i++) {
    cout << shapes[i]->getArea() << endl;  // 动态绑定
}
```

## 七、函数重写与函数重载

### 7.1 函数重写（Override）

在派生类中重新定义基类的虚函数。

```cpp
class Base {
public:
    virtual void show() { cout << "Base\n"; }
};

class Derived : public Base {
public:
    void show() override { cout << "Derived\n"; }
};
```

### 7.2 函数重载（Overload）

在同一类中定义多个同名但参数不同的函数。

```cpp
class MyClass {
public:
    void print(int x) { cout << x << endl; }
    void print(string s) { cout << s << endl; }
};
```

## 八、protected 成员

### 8.1 protected 的作用

protected 成员可以被派生类访问，但不能被外部访问。

```cpp
class Base {
protected:
    int value;
};

class Derived : public Base {
public:
    void setValue(int v) {
        value = v;  // 可以访问 protected 成员
    }
};
```

## 九、多重继承

### 9.1 语法

```cpp
class A { /* ... */ };
class B { /* ... */ };
class C : public A, public B { /* ... */ };
```

### 9.2 菱形继承问题

```cpp
class A { public: int x; };
class B : public A { };
class C : public A { };
class D : public B, public C { };

D d;
// d.x;  // 歧义！B::x 还是 C::x？
d.B::x = 1;
d.C::x = 2;
```

### 9.3 虚拟继承

```cpp
class A { public: int x; };
class B : virtual public A { };
class C : virtual public A { };
class D : public B, public C { };

D d;
d.x = 1;  // 正确，只有一个 x
```

## 十、实战示例

### 示例1：图形计算器

```cpp
#include <iostream>
using namespace std;

class Shape {
public:
    virtual double getArea() = 0;
    virtual double getPerimeter() = 0;
};

class Circle : public Shape {
private:
    double radius;
public:
    Circle(double r) : radius(r) {}
    
    double getArea() override {
        return 3.14159 * radius * radius;
    }
    
    double getPerimeter() override {
        return 2 * 3.14159 * radius;
    }
};

class Rectangle : public Shape {
private:
    double width, height;
public:
    Rectangle(double w, double h) : width(w), height(h) {}
    
    double getArea() override {
        return width * height;
    }
    
    double getPerimeter() override {
        return 2 * (width + height);
    }
};

int main() {
    Shape* shapes[] = {new Circle(5), new Rectangle(4, 6)};
    
    for (auto shape : shapes) {
        cout << "Area: " << shape->getArea() << endl;
        cout << "Perimeter: " << shape->getPerimeter() << endl;
    }
    
    return 0;
}
```

### 示例2：动物模拟器

```cpp
#include <iostream>
using namespace std;

class Animal {
protected:
    string name;
public:
    Animal(string n) : name(n) {}
    virtual void makeSound() = 0;
    virtual void move() = 0;
};

class Dog : public Animal {
public:
    Dog(string n) : Animal(n) {}
    
    void makeSound() override {
        cout << name << " says: Woof!\n";
    }
    
    void move() override {
        cout << name << " runs.\n";
    }
};

class Bird : public Animal {
public:
    Bird(string n) : Animal(n) {}
    
    void makeSound() override {
        cout << name << " says: Chirp!\n";
    }
    
    void move() override {
        cout << name << " flies.\n";
    }
};

int main() {
    Animal* animals[] = {new Dog("Buddy"), new Bird("Tweet")};
    
    for (auto animal : animals) {
        animal->makeSound();
        animal->move();
    }
    
    return 0;
}
```

## 十一、总结

| 概念 | 说明 |
|------|------|
| 继承 | 派生类获取基类的属性和方法 |
| 访问说明符 | public、protected、private |
| 虚函数 | 实现动态多态 |
| 纯虚函数 | 定义抽象接口 |
| 虚析构函数 | 正确释放派生类资源 |
| 多重继承 | 一个类继承多个基类 |
| 虚拟继承 | 解决菱形继承问题 |

继承和多态是面向对象编程的核心，掌握它们可以设计出更加灵活和可扩展的代码。
