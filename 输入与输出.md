
# C++ 输入与输出

## 一、概述

C++ 的输入与输出（I/O）操作主要通过标准库提供的流来实现。标准库定义了多个流对象和操作符，使得数据的输入输出变得简单直观。

### 1.1 标准流对象

C++ 标准库提供了四个预定义的流对象：

| 流对象 | 含义 | 关联设备 |
|--------|------|----------|
| `cin` | 标准输入 | 键盘 |
| `cout` | 标准输出 | 显示器 |
| `cerr` | 标准错误 | 显示器 |
| `clog` | 标准日志 | 显示器 |

### 1.2 头文件

使用标准 I/O 需要包含头文件：

```cpp
#include <iostream>
```

## 二、基本输出操作

### 2.1 使用 cout 输出

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    cout << "Value is: " << 42 << endl;
    return 0;
}
```

### 2.2 输出操作符 `<<`

`<<` 称为插入操作符，可以串联使用：

```cpp
int a = 10;
double b = 3.14;
cout << "a = " << a << ", b = " << b << endl;
```

### 2.3 换行符

有两种方式可以实现换行：

```cpp
cout << "Line 1\n";           // 使用转义字符
cout << "Line 2" << endl;     // 使用 endl 操纵符
```

**区别**：
- `\n` 仅输出换行符
- `endl` 输出换行符并刷新缓冲区

## 三、基本输入操作

### 3.1 使用 cin 输入

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "Enter a number: ";
    cin >> num;
    cout << "You entered: " << num << endl;
    return 0;
}
```

### 3.2 输入操作符 `>>`

`>>` 称为提取操作符，同样可以串联使用：

```cpp
int x, y;
cout << "Enter two numbers: ";
cin >> x >> y;
cout << "Sum: " << x + y << endl;
```

### 3.3 输入字符串

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cout << "Enter your name: ";
    cin >> name;  // 读取到空格为止
    cout << "Hello, " << name << "!" << endl;
    return 0;
}
```

## 四、格式化输出

### 4.1 设置输出宽度

```cpp
#include <iostream>
#include <iomanip>  // 需要包含此头文件
using namespace std;

int main() {
    cout << setw(10) << "Name" << setw(10) << "Age" << endl;
    cout << setw(10) << "Alice" << setw(10) << 25 << endl;
    return 0;
}
```

### 4.2 设置小数精度

```cpp
double pi = 3.1415926535;
cout << "Default: " << pi << endl;
cout << "Fixed: " << fixed << setprecision(2) << pi << endl;
cout << "Scientific: " << scientific << setprecision(3) << pi << endl;
```

### 4.3 设置对齐方式

```cpp
cout << left << setw(10) << "Left" << endl;
cout << right << setw(10) << "Right" << endl;
cout << internal << setw(10) << -123 << endl;
```

### 4.4 进制转换

```cpp
int num = 42;
cout << "Decimal: " << dec << num << endl;   // 十进制
cout << "Octal: " << oct << num << endl;     // 八进制
cout << "Hex: " << hex << num << endl;       // 十六进制
cout << "Hex uppercase: " << uppercase << hex << num << endl;
```

## 五、字符输入输出

### 5.1 单个字符输入

```cpp
char ch;
cin >> ch;           // 跳过空白字符
cin.get(ch);         // 读取任意字符（包括空格）
ch = cin.get();      // 读取字符并返回
```

### 5.2 忽略字符

```cpp
cin.ignore();        // 忽略一个字符
cin.ignore(10);      // 忽略最多10个字符
cin.ignore(numeric_limits<streamsize>::max(), '\n');  // 忽略整行
```

## 六、字符串输入输出

### 6.1 读取整行

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string line;
    cout << "Enter a line: ";
    getline(cin, line);  // 读取整行，包括空格
    cout << "You entered: " << line << endl;
    return 0;
}
```

### 6.2 混合输入注意事项

当混合使用 `cin >>` 和 `getline` 时需要注意：

```cpp
int age;
string name;

cin >> age;
cin.ignore();          // 忽略换行符
getline(cin, name);    // 现在可以正确读取
```

## 七、文件输入输出

### 7.1 写入文件

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream outfile("example.txt");
    if (outfile.is_open()) {
        outfile << "Hello, File!" << endl;
        outfile << "Number: " << 42 << endl;
        outfile.close();
    } else {
        cout << "Unable to open file" << endl;
    }
    return 0;
}
```

### 7.2 读取文件

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ifstream infile("example.txt");
    string line;
    if (infile.is_open()) {
        while (getline(infile, line)) {
            cout << line << endl;
        }
        infile.close();
    } else {
        cout << "Unable to open file" << endl;
    }
    return 0;
}
```

### 7.3 文件操作模式

| 模式 | 含义 |
|------|------|
| `ios::in` | 读取模式（默认 for ifstream） |
| `ios::out` | 写入模式（默认 for ofstream） |
| `ios::app` | 追加模式 |
| `ios::binary` | 二进制模式 |
| `ios::trunc` | 截断模式（覆盖原有内容） |
| `ios::ate` | 定位到文件末尾 |

```cpp
ofstream outfile("data.txt", ios::out | ios::app);  // 追加模式
```

## 八、错误处理

### 8.1 检查流状态

```cpp
int num;
cin >> num;
if (cin.fail()) {
    cout << "Input error!" << endl;
    cin.clear();  // 清除错误标志
    cin.ignore(numeric_limits<streamsize>::max(), '\n');  // 忽略错误输入
}
```

### 8.2 流状态标志

| 标志 | 含义 |
|------|------|
| `good()` | 流状态正常 |
| `eof()` | 到达文件末尾 |
| `fail()` | 输入/输出操作失败 |
| `bad()` | 流发生严重错误 |

## 九、实战示例

### 示例1：简单计算器

```cpp
#include <iostream>
using namespace std;

int main() {
    double num1, num2;
    char op;
    
    cout << "Enter expression (e.g., 3 + 4): ";
    cin >> num1 >> op >> num2;
    
    switch(op) {
        case '+': cout << num1 + num2 << endl; break;
        case '-': cout << num1 - num2 << endl; break;
        case '*': cout << num1 * num2 << endl; break;
        case '/': 
            if (num2 != 0) cout << num1 / num2 << endl;
            else cout << "Division by zero!" << endl;
            break;
        default: cout << "Invalid operator" << endl;
    }
    return 0;
}
```

### 示例2：文件复制

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ifstream source("source.txt", ios::binary);
    ofstream dest("dest.txt", ios::binary);
    
    if (source && dest) {
        dest << source.rdbuf();  // 直接复制缓冲区
        cout << "File copied successfully" << endl;
    } else {
        cout << "Error copying file" << endl;
    }
    return 0;
}
```

## 十、总结

| 操作类型 | 工具 | 用途 |
|----------|------|------|
| 标准输出 | `cout` | 向屏幕输出数据 |
| 标准输入 | `cin` | 从键盘读取数据 |
| 文件输出 | `ofstream` | 向文件写入数据 |
| 文件输入 | `ifstream` | 从文件读取数据 |
| 格式化 | `<iomanip>` | 控制输出格式 |

C++ 的 I/O 系统提供了丰富的功能，掌握基本的输入输出操作是学习 C++ 的重要基础。
