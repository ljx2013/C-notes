
# C++ 类与对象

## 一、面向对象编程概述

面向对象编程（OOP）是一种编程范式，将数据和操作数据的方法封装在一起。核心概念包括：类、对象、封装、继承和多态。

### 1.1 类与对象的关系

- **类**：定义对象的蓝图或模板
- **对象**：类的实例

## 二、类的定义

### 2.1 基本语法

```cpp
class ClassName {
public:
    // 公有成员（外部可访问）
private:
    // 私有成员（仅类内部可访问）
protected:
    // 受保护成员（类内部和派生类可访问）
};
```

### 2.2 示例：定义一个简单类

```cpp
class Person {
private:
    std::string name;
    int age;

public:
    // 成员函数
    void setName(std::string n) {
        name = n;
    }
    
    std::string getName() {
        return name;
    }
    
    void setAge(int a) {
        if (a >= 0) {
            age = a;
        }
    }
    
    int getAge() {
        return age;
    }
};
```

## 三、对象的创建与使用

### 3.1 创建对象

```cpp
Person p;                    // 栈上创建
Person* p = new Person();    // 堆上创建
```

### 3.2 访问成员

```cpp
Person p;
p.setName("Alice");
p.setAge(25);

std::cout << p.getName() << " is " << p.getAge() << " years old";
```

## 四、构造函数与析构函数

### 4.1 构造函数

构造函数在对象创建时自动调用，用于初始化对象。

```cpp
class Person {
private:
    std::string name;
    int age;

public:
    // 默认构造函数
    Person() {
        name = "Unknown";
        age = 0;
    }
    
    // 带参数的构造函数
    Person(std::string n, int a) {
        name = n;
        age = a;
    }
};
```

### 4.2 析构函数

析构函数在对象销毁时自动调用，用于清理资源。

```cpp
class Person {
public:
    ~Person() {
        // 清理资源
    }
};
```

### 4.3 构造函数初始化列表

```cpp
class Person {
private:
    std::string name;
    int age;

public:
    Person(std::string n, int a) : name(n), age(a) {
        // 初始化列表
    }
};
```

## 五、封装

### 5.1 访问控制

| 访问修饰符 | 类内部 | 派生类 | 外部 |
|------------|--------|--------|------|
| public | 是 | 是 | 是 |
| protected | 是 | 是 | 否 |
| private | 是 | 否 | 否 |

### 5.2 封装的优点

- 数据隐藏：保护数据不被非法访问
- 接口简化：只暴露必要的方法
- 易于维护：修改内部实现不影响外部

## 六、静态成员

### 6.1 静态成员变量

属于类而不是对象，所有对象共享。

```cpp
class Counter {
private:
    static int count;  // 静态成员变量

public:
    Counter() {
        count++;
    }
    
    static int getCount() {
        return count;
    }
};

// 静态成员变量必须在类外初始化
int Counter::count = 0;
```

### 6.2 静态成员函数

不依赖于对象，可以直接通过类名调用。

```cpp
Counter::getCount();  // 直接调用
```

## 七、友元

### 7.1 友元函数

允许外部函数访问类的私有成员。

```cpp
class Box {
private:
    double width;
    
public:
    Box(double w) : width(w) {}
    
    friend double getWidth(Box box);
};

double getWidth(Box box) {
    return box.width;  // 可以访问私有成员
}
```

### 7.2 友元类

允许另一个类访问本类的私有成员。

```cpp
class Box {
private:
    double width;
    
public:
    Box(double w) : width(w) {}
    
    friend class BoxPrinter;
};

class BoxPrinter {
public:
    void printWidth(Box box) {
        std::cout << box.width;  // 可以访问私有成员
    }
};
```

## 八、this 指针

### 8.1 this 指针的使用

`this` 是一个指向当前对象的指针。

```cpp
class Person {
private:
    std::string name;

public:
    void setName(std::string name) {
        this->name = name;  // 区分参数和成员变量
    }
};
```

## 九、对象数组

```cpp
class Person {
private:
    std::string name;

public:
    Person(std::string n = "Unknown") : name(n) {}
    
    std::string getName() {
        return name;
    }
};

int main() {
    Person people[3] = {Person("Alice"), Person("Bob"), Person("Charlie")};
    
    for (int i = 0; i < 3; i++) {
        std::cout << people[i].getName() << std::endl;
    }
    
    return 0;
}
```

## 十、实战示例

### 示例1：实现一个学生类

```cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
private:
    string name;
    int id;
    double gpa;

public:
    Student(string n, int i, double g) : name(n), id(i), gpa(g) {}
    
    void display() {
        cout << "Name: " << name << endl;
        cout << "ID: " << id << endl;
        cout << "GPA: " << gpa << endl;
    }
    
    bool isHonors() {
        return gpa >= 3.5;
    }
};

int main() {
    Student s("Alice", 12345, 3.8);
    s.display();
    
    if (s.isHonors()) {
        cout << "Honors student!" << endl;
    }
    
    return 0;
}
```

### 示例2：实现一个银行账户类

```cpp
#include <iostream>
#include <string>
using namespace std;

class BankAccount {
private:
    string accountNumber;
    double balance;

public:
    BankAccount(string num, double bal) : accountNumber(num), balance(bal) {}
    
    void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
        }
    }
    
    bool withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            return true;
        }
        return false;
    }
    
    double getBalance() {
        return balance;
    }
};

int main() {
    BankAccount account("123456", 1000.0);
    account.deposit(500.0);
    account.withdraw(200.0);
    cout << "Balance: " << account.getBalance() << endl;
    return 0;
}
```

## 十一、总结

| 概念 | 说明 |
|------|------|
| 类 | 对象的蓝图，定义属性和方法 |
| 对象 | 类的实例 |
| 构造函数 | 初始化对象 |
| 析构函数 | 清理资源 |
| 封装 | 数据隐藏和接口简化 |
| 静态成员 | 属于类，所有对象共享 |
| 友元 | 允许外部访问私有成员 |
| this | 指向当前对象的指针 |

类和对象是面向对象编程的基础，掌握它们对于编写模块化、可维护的代码至关重要。
