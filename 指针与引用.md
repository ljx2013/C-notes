
# C++ 指针与引用

## 一、指针概述

指针是一个变量，其值为另一个变量的内存地址。指针允许我们直接操作内存，是C++中非常强大的特性。

### 1.1 内存地址

```cpp
int x = 10;
cout << &x;  // 输出变量x的内存地址
```

### 1.2 指针变量声明

```cpp
int* ptr;    // 指向int类型的指针
char* cptr;  // 指向char类型的指针
double* dptr; // 指向double类型的指针
```

### 1.3 指针的基本操作

```cpp
int x = 42;
int* ptr = &x;  // 取地址

*ptr = 100;     // 解引用，修改x的值
cout << *ptr;   // 解引用，读取x的值
```

## 二、指针运算

### 2.1 指针算术

```cpp
int arr[] = {10, 20, 30, 40, 50};
int* p = arr;   // 数组名是首元素地址

cout << *p;     // 10
cout << *(p+1); // 20
cout << *(p+2); // 30
```

### 2.2 指针比较

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int* p = arr;
int* q = arr + 3;

if (p < q) {
    cout << "p is before q";
}
```

### 2.3 指针与数组

```cpp
int arr[5] = {10, 20, 30, 40, 50};

// 数组下标方式
cout << arr[2];  // 30

// 指针方式
int* p = arr;
cout << *(p + 2); // 30
```

## 三、动态内存分配

### 3.1 使用 new 分配内存

```cpp
// 分配单个变量
int* p = new int;
*p = 42;

// 分配数组
int* arr = new int[5];
for (int i = 0; i < 5; i++) {
    arr[i] = i + 1;
}
```

### 3.2 使用 delete 释放内存

```cpp
// 释放单个变量
delete p;

// 释放数组
delete[] arr;
```

### 3.3 空指针检查

```cpp
int* p = new (nothrow) int;
if (p == nullptr) {
    cout << "Memory allocation failed";
}
```

## 四、引用

### 4.1 引用的定义

引用是变量的别名，必须在声明时初始化。

```cpp
int x = 10;
int& ref = x;  // ref是x的引用

ref = 20;      // 修改ref，x也被修改
cout << x;     // 输出20
```

### 4.2 引用与指针的区别

| 特性 | 指针 | 引用 |
|------|------|------|
| 空值 | 可以为空 | 不能为空 |
| 重新赋值 | 可以指向其他变量 | 始终指向初始变量 |
| 操作符 | 使用 * 和 -> | 直接使用 |
| 内存开销 | 占用内存 | 不占用额外内存 |

### 4.3 引用的用途

#### 作为函数参数

```cpp
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

// 使用
int x = 5, y = 10;
swap(x, y);
```

#### 作为函数返回值

```cpp
int& getElement(int arr[], int index) {
    return arr[index];
}

// 使用
int arr[5] = {1, 2, 3, 4, 5};
getElement(arr, 2) = 100;  // 修改第三个元素
```

## 五、指针与引用的高级用法

### 5.1 指向指针的指针

```cpp
int x = 10;
int* p = &x;
int** pp = &p;

cout << **pp;  // 输出10
**pp = 20;     // 修改x的值
```

### 5.2 指针数组

```cpp
int a = 1, b = 2, c = 3;
int* arr[3] = {&a, &b, &c};

cout << *arr[0];  // 输出1
cout << *arr[1];  // 输出2
```

### 5.3 数组指针

```cpp
int arr[3][4];
int (*p)[4] = arr;  // p指向包含4个int的数组

p++;  // 移动到下一行
```

## 六、const 与指针/引用

### 6.1 const 指针

```cpp
int x = 10;

// 指向const的指针（指针指向的值不可修改）
const int* p = &x;
// *p = 20;  // 错误

// const指针（指针本身不可修改）
int* const p = &x;
// p = &y;  // 错误

// 指向const的const指针
const int* const p = &x;
```

### 6.2 const 引用

```cpp
int x = 10;
const int& ref = x;
// ref = 20;  // 错误
```

## 七、智能指针

### 7.1 unique_ptr

独占所有权的智能指针。

```cpp
#include <memory>

std::unique_ptr<int> p = std::make_unique<int>(42);
std::unique_ptr<int[]> arr = std::make_unique<int[]>(5);
```

### 7.2 shared_ptr

共享所有权的智能指针。

```cpp
std::shared_ptr<int> p1 = std::make_shared<int>(10);
std::shared_ptr<int> p2 = p1;  // 引用计数增加
```

### 7.3 weak_ptr

弱引用智能指针，不增加引用计数。

```cpp
std::weak_ptr<int> wp = p1;
if (auto sp = wp.lock()) {
    // 使用sp
}
```

## 八、常见陷阱

### 8.1 野指针

```cpp
int* p;           // 未初始化的指针（野指针）
*p = 10;          // 危险！未定义行为
```

### 8.2 悬垂指针

```cpp
int* p = new int(10);
delete p;
*p = 20;          // 危险！指针已释放
```

### 8.3 空指针解引用

```cpp
int* p = nullptr;
*p = 10;          // 错误！空指针解引用
```

## 九、实战示例

### 示例1：使用指针反转数组

```cpp
#include <iostream>
using namespace std;

void reverse(int* arr, int size) {
    int* start = arr;
    int* end = arr + size - 1;
    
    while (start < end) {
        int temp = *start;
        *start = *end;
        *end = temp;
        start++;
        end--;
    }
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int size = sizeof(arr) / sizeof(arr[0]);
    
    reverse(arr, size);
    
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}
```

### 示例2：动态二维数组

```cpp
#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 4;
    
    // 分配行指针
    int** matrix = new int*[rows];
    
    // 分配每行的数据
    for (int i = 0; i < rows; i++) {
        matrix[i] = new int[cols];
    }
    
    // 填充数据
    int count = 1;
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            matrix[i][j] = count++;
        }
    }
    
    // 输出
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++) {
            cout << matrix[i][j] << " ";
        }
        cout << endl;
    }
    
    // 释放内存
    for (int i = 0; i < rows; i++) {
        delete[] matrix[i];
    }
    delete[] matrix;
    
    return 0;
}
```

## 十、总结

| 概念 | 说明 |
|------|------|
| 指针 | 存储变量地址的变量 |
| 解引用 | 使用 * 访问指针指向的值 |
| 引用 | 变量的别名，必须初始化 |
| 动态内存 | 使用 new/delete 管理 |
| 智能指针 | 自动管理内存，避免泄漏 |
| const指针 | 保护指针指向的值或指针本身 |

指针和引用是C++的核心概念，掌握它们对于编写高效、灵活的代码至关重要。
