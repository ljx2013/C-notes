
# C++ 函数与作用域

## 一、函数概述

函数是一组执行特定任务的代码块，通过函数名调用执行。函数可以接收参数、返回值，实现代码复用和模块化。

### 1.1 函数的组成

```cpp
返回类型 函数名(参数列表) {
    // 函数体
    return 返回值;
}
```

### 1.2 函数声明与定义

**声明**（函数原型）：
```cpp
int add(int a, int b);  // 声明
```

**定义**（实现）：
```cpp
int add(int a, int b) {
    return a + b;
}
```

## 二、函数参数

### 2.1 参数传递方式

#### 值传递
```cpp
void swap(int a, int b) {
    int temp = a;
    a = b;
    b = temp;  // 只影响局部变量
}
```

#### 引用传递
```cpp
void swap(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;  // 影响实参
}
```

#### 指针传递
```cpp
void swap(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;  // 通过指针修改实参
}
```

### 2.2 默认参数

```cpp
int multiply(int a, int b = 2) {
    return a * b;
}

// 调用
multiply(5);    // 5 * 2 = 10
multiply(5, 3); // 5 * 3 = 15
```

### 2.3 可变参数

```cpp
#include <cstdarg>

int sum(int count, ...) {
    va_list args;
    va_start(args, count);
    
    int total = 0;
    for (int i = 0; i < count; i++) {
        total += va_arg(args, int);
    }
    
    va_end(args);
    return total;
}

// 调用
sum(3, 1, 2, 3);  // 返回 6
```

## 三、函数返回值

### 3.1 返回单个值

```cpp
int max(int a, int b) {
    return (a > b) ? a : b;
}
```

### 3.2 返回引用

```cpp
int& getMax(int& a, int& b) {
    return (a > b) ? a : b;
}

// 使用
int x = 10, y = 20;
getMax(x, y) = 30;  // 修改较大的值
```

### 3.3 返回多个值

使用引用参数：
```cpp
void divide(int a, int b, int& quotient, int& remainder) {
    quotient = a / b;
    remainder = a % b;
}
```

使用结构体：
```cpp
struct Result {
    int quotient;
    int remainder;
};

Result divide(int a, int b) {
    Result res;
    res.quotient = a / b;
    res.remainder = a % b;
    return res;
}
```

## 四、函数重载

### 4.1 概念

同一作用域内，函数名相同但参数列表不同的多个函数。

### 4.2 示例

```cpp
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

int add(int a, int b, int c) {
    return a + b + c;
}
```

### 4.3 重载解析

编译器根据参数类型和数量选择合适的函数。

## 五、递归函数

### 5.1 概念

函数调用自身的编程技巧。

### 5.2 示例：阶乘

```cpp
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;  // 基例
    }
    return n * factorial(n - 1);  // 递归调用
}
```

### 5.3 示例：斐波那契数列

```cpp
int fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}
```

### 5.4 尾递归优化

```cpp
int factorialTail(int n, int acc = 1) {
    if (n == 0) {
        return acc;
    }
    return factorialTail(n - 1, n * acc);
}
```

## 六、内联函数

### 6.1 概念

编译时将函数体插入调用处，减少函数调用开销。

### 6.2 语法

```cpp
inline int square(int x) {
    return x * x;
}
```

### 6.3 使用场景

- 函数体短小
- 被频繁调用
- 无复杂控制流

## 七、作用域

### 7.1 作用域类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 全局作用域 | 整个程序可见 | 全局变量 |
| 局部作用域 | 函数/块内可见 | 局部变量 |
| 类作用域 | 类内部可见 | 成员变量 |
| 命名空间作用域 | 命名空间内可见 | 命名空间成员 |

### 7.2 变量生命周期

```cpp
#include <iostream>
using namespace std;

int global = 10;  // 全局变量，程序生命周期

void func() {
    static int count = 0;  // 静态局部变量，函数第一次调用时初始化
    count++;
    cout << count << endl;
}

int main() {
    func();  // 输出 1
    func();  // 输出 2
    func();  // 输出 3
    return 0;
}
```

### 7.3 作用域解析运算符

```cpp
int x = 10;  // 全局变量

int main() {
    int x = 5;  // 局部变量
    cout << x;        // 输出 5（局部）
    cout << ::x;      // 输出 10（全局）
    return 0;
}
```

## 八、命名空间

### 8.1 定义命名空间

```cpp
namespace myspace {
    int add(int a, int b) {
        return a + b;
    }
}
```

### 8.2 使用命名空间

```cpp
// 方式1：使用作用域解析
myspace::add(3, 4);

// 方式2：using声明
using myspace::add;
add(3, 4);

// 方式3：using指令
using namespace myspace;
add(3, 4);
```

### 8.3 嵌套命名空间

```cpp
namespace outer {
    namespace inner {
        int value = 42;
    }
}

// 访问
outer::inner::value;
```

## 九、实战示例

### 示例1：计算器函数

```cpp
#include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
int subtract(int a, int b) { return a - b; }
int multiply(int a, int b) { return a * b; }
int divide(int a, int b) { 
    if (b == 0) {
        cout << "Division by zero!" << endl;
        return 0;
    }
    return a / b;
}

int main() {
    int choice, a, b;
    cout << "1. Add\n2. Subtract\n3. Multiply\n4. Divide\n";
    cin >> choice >> a >> b;
    
    switch(choice) {
        case 1: cout << add(a, b); break;
        case 2: cout << subtract(a, b); break;
        case 3: cout << multiply(a, b); break;
        case 4: cout << divide(a, b); break;
        default: cout << "Invalid choice";
    }
    return 0;
}
```

### 示例2：递归二分查找

```cpp
#include <iostream>
using namespace std;

int binarySearch(int arr[], int left, int right, int target) {
    if (right >= left) {
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) {
            return mid;
        }
        
        if (arr[mid] > target) {
            return binarySearch(arr, left, mid - 1, target);
        }
        
        return binarySearch(arr, mid + 1, right, target);
    }
    
    return -1;  // 未找到
}

int main() {
    int arr[] = {2, 5, 8, 12, 16, 23, 38, 56, 72, 91};
    int n = sizeof(arr) / sizeof(arr[0]);
    int target = 23;
    
    int result = binarySearch(arr, 0, n - 1, target);
    if (result != -1) {
        cout << "Element found at index " << result;
    } else {
        cout << "Element not found";
    }
    return 0;
}
```

## 十、总结

| 概念 | 说明 |
|------|------|
| 函数声明 | 告诉编译器函数的名字、返回类型和参数 |

| 参数传递 | 值传递、引用传递、指针传递 |
